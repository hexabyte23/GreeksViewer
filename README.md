# GreeksViewer
Tool for pricing financial derivative products using Black Scholes formulas.

## Derivative products priced:

- Option on stock with dividend
	- Premium	(3d/3d)
	- Delta		(3d/3d)
	- Gamma		(3d/3d)
	- Theta		(3d/3d)
	- Rho		(3d/3d)
	- Vega		(3d/3d)
- Forward
	- Theta		(3d/3d)
	- Rho	 	(3d/3d)
- Straddle on stock with dividend
	- Premium	(3d/3d)
	- Delta		(3d/3d)
	- Gamma		(3d/3d)
	- Theta		(3d/3d)
	- Rho		(3d/3d)
	- Vega		(3d/3d)
- Strangle on stock with dividend
	- Premium	(3d/3d)
	- Delta		(3d/3d)
	- Gamma		(3d/3d)
	- Theta		(3d/3d)
	- Rho		(3d/3d)
	- Vega		(3d/3d)


# Installation

Download last version of gnuplot at http://www.gnuplot.info

Open a terminal session

```
cd src/gnuplot 

gnuplot bs-2d.txt
```
or
```
gnuplot bs-3d.txt
```
